# football-league-manager

# 축구 리그 승점 관리 프로그램

## 프로그램 소개

축구 리그의 팀 정보를 입력받아 승점과 순위를 계산하는 프로그램이다.

사용자는 리그 이름과 각 팀의 승, 무, 패 기록을 입력할 수 있으며, 프로그램은 경기 수와 승점을 자동으로 계산하여 순위표를 출력한다.

---

## 주요 기능

* 리그 이름 입력
* 팀 정보 입력
* 경기 수 자동 계산
* 승점 자동 계산
* 승점 기준 순위 정렬
* 동점 시 승 수를 기준으로 정렬
* 순위표 출력

---

## 승점 계산 방식

* 승리 : 3점
* 무승부 : 1점
* 패배 : 0점

승점 계산식

```python
points = win * 3 + draw
```

---

## 코드 설명

### 1. 팀 정보 입력

사용자로부터 팀 이름과 승, 무, 패 기록을 입력받는다.

```python
name = input("팀 이름: ")
win = int(input("승: "))
draw = int(input("무: "))
lose = int(input("패: "))
```

### 2. 경기 수 및 승점 계산

입력받은 승, 무, 패를 이용하여 경기 수와 승점을 계산한다.

```python
games = win + draw + lose
points = win * 3 + draw
```

### 3. 팀 정보 저장

각 팀의 정보를 딕셔너리 형태로 저장한 후 리스트에 추가한다.

```python
team = {
    "name": name,
    "games": games,
    "win": win,
    "draw": draw,
    "lose": lose,
    "points": points
}
```

### 4. 순위 정렬

승점이 높은 순서대로 정렬하며, 승점이 같을 경우 승 수가 많은 팀을 상위에 배치한다.

```python
teams.sort(
    key=lambda x: (x["points"], x["win"]),
    reverse=True
)
```

### 5. 순위표 출력

정렬된 결과를 이용하여 최종 순위를 출력한다.

```python
for i, team in enumerate(teams):
```
